# quant

We would be having some data and would run some models on it for making some decisions.
This is a part of Trading Straytegies and Algo Trading.

All that we do would be a combination of mathematical concepts and models.

We will also touch upon mathematical concepts like:
- Linear Algebra: Vectors=Returns, Matrix=Covariance, EigenValues=Risk
- Probability theory: Random Variables=Returns, Avg Return=Expectation, Variance=Risk (Normal Distribution, fat tails)
- Stochastic Calculus: Prices follow random walks with drift and noise. This drives Monte Carlo, Black Scholes
- Convex Optimization: Used in Portfolio optimization, risk minimization, finding weights that maximise Sharpe

We will use models like:
- Momentum Models: Uses SMA, EMA, Trends often follow, reversals are tough. General Rules.
- Volatility Models: How much risk to take, automatic adjustment to protect postfolios during market stress.
- Rate and Macro Models: Asset Rotation, Cycles, models shift allocation from long durations assets.
- Correlation models: Manage risks, prevent correlation when markets become unstable.
- Regime Models: Markets move through phases like risk-on, risk-off, inflationary, recessionary, Yield curves.

Aspects used:
- Machine learning: linear regression, random forest, xgboost, lstm
- Price Derivative using mathematical models: Black Scholes, binomial tree, trinomial tree, monte carlo simulation (intuition for pricing, risk)
- Value at Risk (VaR model): Historical Var, Variance-covariance, monte carlo Var)
- Credit Risk modelling: logistic regression, scorecard modelling, PG-LGD-EAD frameworks, BASEL, IFRS9, CECL, CCAR
- Volatility based models: GARCH, EGARCH, GJR-GARCH understand how momentum impacts market
